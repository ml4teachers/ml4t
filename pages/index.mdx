# Willkommen bei Machine Learning for Teachers

**ML4T** ist eine Ressource, die speziell für Lehrkräfte entwickelt wurde, um das Verständnis von **maschinellem Lernen** (ML) mithilfe interaktiver Übungen zu fördern.

Die Webseite ist in zwei Übungen unterteilt: **Song Recommender** und **Movie Recommender**. Beide Übungen demonstrieren, wie maschinelles Lernen dazu verwendet werden kann, personalisierte Empfehlungen zu erstellen – eine Anwendung, die in vielen der heute genutzten digitalen Dienste zu finden ist.

## Song Recommender
In der Übung [Song Recommender](/song-recommender) wird aufgezeigt, wie man einen Algorithmus entwickelt, der auf Basis von demografischen Merkmalen und persönlichen Vorlieben individuelle Songempfehlungen abgeben kann. Hier erfährst du, wie Daten von Einzelpersonen genutzt werden können, um basierend auf spezifischen Benutzereingaben personalisierte Songempfehlungen zu erstellen.
